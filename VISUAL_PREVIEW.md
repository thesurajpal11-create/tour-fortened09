## ⚙️ Technology Used

**Backend:**
- FastAPI (Modern Python web framework)
- MySQL (Database)
- SQLAlchemy (ORM)
- Pydantic (Data validation)
- Bcrypt (Password security)

**Frontend:**
- HTML5 (Structure)
- CSS3 (Styling)
- JavaScript ES6+ (Interactivity)
- Fetch API (API calls)
